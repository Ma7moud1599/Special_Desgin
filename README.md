# Special Design — CSS Showcase

A collection of creative CSS designs and UI components. Demonstrates advanced CSS techniques including animations, gradients, glassmorphism, and responsive layouts.

## Includes

- Animated hero sections
- CSS-only card effects (hover, 3D transform)
- Gradient backgrounds and glassmorphism cards
- Custom button styles and micro-interactions
- Responsive navigation layouts

## Running

Open any `.html` file in a browser — pure HTML/CSS, no JavaScript dependencies.

## Tech Stack

HTML5, CSS3 (animations, transforms, custom properties, grid, flexbox)
